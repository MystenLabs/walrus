Overview of guides for setting up and operating a Walrus storage node.

This section covers everything you need to set up and operate a Walrus storage node.

- [Backup and Restore Guide](/docs/operator-guide/storage-nodes/backup-restore-guide): Back up and restore Walrus storage node databases with automated and manual procedures.
- [Commission and Governance](/docs/operator-guide/storage-nodes/commission-governance): Manage storage node commission collection and governance operations including authorization and contract upgrades.
- [Slashing Walkthrough](/docs/operator-guide/storage-nodes/slashing-walkthrough): A worked example of a slashing event on Mainnet, with the concrete vote, execute, and cleanup commands operators run end to end.
- [Slashing](/docs/operator-guide/storage-nodes/slashing): Vote to slash misbehaving storage nodes through quorum-based governance that burns the offender's accumulated commission.
- [Storage Node FAQ](/docs/operator-guide/storage-nodes/storage-node-faq): Frequently asked questions about operating a Walrus storage node, including wallets, TLS, RPC configuration, and commission rates.
- [Storage Node Maintenance](/docs/operator-guide/storage-nodes/storage-node-maintenance): Monitor, update, and maintain a Walrus storage node, including key metrics, update procedures, and community tools.
- [Migrate a Storage Node](/docs/operator-guide/storage-nodes/storage-node-migration): Guide for migrating a Walrus storage node to new hardware with minimal downtime.
- [Set Up a Storage Node](/docs/operator-guide/storage-nodes/storage-node-setup): Step-by-step guide for setting up a Walrus storage node, including system preparation, TLS, configuration, and registration.
